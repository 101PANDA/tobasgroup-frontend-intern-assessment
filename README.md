# Tobams Group - Frontend Assessment

This project is a static page implementation for the Tobams Group assessment, built using Next.js and Tailwind CSS.

## Figma Design

## Stack

- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Setup

1. Clone the repository: `git clone https://github.com/101PANDA/tobasgroup-frontend-intern-assessment.git`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Design Decisions & Assumptions

- **Assumptions:** I assumed the dropdown menus in the nav are placeholders and do not require functional state for this static assessment.
- **Customizations:** To maintain a clean component-based architecture, I defined some typography styles based on the provided Figma design.
